# Homebrew Tap for Amos

This is a custom Homebrew tap for [Amos](https://github.com/apodacaa/amos), a minimal TUI for journal + todo management with brutalist design.

## Installation

```bash
brew tap apodacaa/amos
brew install amos
```

## Usage

After installation, run:

```bash
amos
```

## Updating

To upgrade to the latest version:

```bash
brew update
brew upgrade amos
```

## Uninstalling

```bash
brew uninstall amos
brew untap apodacaa/amos
```
